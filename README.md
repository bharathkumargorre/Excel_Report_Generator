# Excel Report Generator

A Python-based Excel Report Generator that automates sales analysis and creates professional Excel reports with charts and dashboards.

## Features
* Generate Excel reports automatically
* Calculate total revenue
* Category-wise sales summary
* Monthly sales analysis
* Top-selling products analysis
* Dashboard sheet
* Bar charts
* Line charts
* Pie charts
* Professional formatting using OpenPyXL

## Technologies Used
* Python
* Pandas
* OpenPyXL

## Installation
```bash
pip install -r requirements.txt
```

## Run
```bash
python app.py
```

## Output
The application generates timestamped Excel reports containing:

* Sales Data
* Category Summary
* Monthly Summary
* Top Products
* Dashboard
